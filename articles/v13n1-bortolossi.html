Abstract

This text explores, from a numerical perspective, the use of the affine function l(x) = x+1 to approximate the function f(x) = exp(x), including values of x not so close to p = 0. The exposition is visually and graphically implemented in GeoGebra, allowing for an interactive understanding of the approximation.
