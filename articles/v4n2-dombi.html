Using GeoGebra to Present Kinetic Data and Ligand Binding Data to a Biochemistry Class

George Dombi, John Golden
George Dombi — University of Rhode Island
John Golden — Grand Valley State University
⬇ Download PDF

Abstract

There are two hyperbolic functions commonly presented in Biochemistry classes. Both of these functions, the Michaelis-Menten enzyme activity curve and the Scatchard ligand binding isotherm, are routinely re-expressed using a linear transform in order to determine the rate and binding constants inherent in each model. We will demonstrate how these transforms, when presented in GeoGebra, make useful and interactive teaching tools.&nbsp;

Keywords: hyperbolic functions, Biochemistry, Scatchard ligand binding isotherm
